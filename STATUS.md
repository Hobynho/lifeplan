## 🎯 Para Expandir

### Adicionar Página
```bash
# Crie a pasta
mkdir -p app/nova-pagina

# Crie o arquivo
touch app/nova-pagina/page.tsx

# Use os componentes existentes
```

### Criar Novo Componente
1. Siga padrão em `QUICK-START.md`
2. [Use design tokens](README-DESIGN-SYSTEM.md#-design-tokens)
3. Exporte em `index.ts`
4. Documente em `COMPONENTS-REFERENCE.md`

### Customizar Design
Edit `tailwind.config.ts`:
```ts
colors: {
  primary: { 500: '#seu-cor' }
}
```
